***Launch Template***
  A Launch Template is a reusable configuration blueprint for launching EC2 instances.
  It can store:AMI. Instance type. Key pair,Security groups, Subnet, User data (startup scripts), IAM role, Tags, Storage (EBS) and Network settings
  Instead of configuring EC2 again and again, you define once and reuse.

***Benefits***
  Reusability
  Versions
  Ease of use and Manage
  Customization for Auto Scaling
