***Example 1: ***

***Host a Simple HTML Website on a Linux EC2 Instance***
```text
Steps to Host a Simple HTML Website on a Linux EC2 Instance:

1. Launch an EC2 Instance
   - Choose Amazon Linux 2 AMI
   - Select instance type (t2.micro for free tier)
   - Configure key pair and security group
   - Allow inbound HTTP (port 80) and SSH (port 22)

2. Connect to the EC2 Instance
   - Use SSH with the key pair to connect to the instance

3. Install Web Server (Apache)
   - Update packages yum update -y 
   - Install Apache (httpd) yum install httpd -y
   - Start and enable the Apache service
      - systemctl start httpd
      - systemctl status httpd
      - chkconfig httpd on

4. Create HTML File
   - Navigate to  cd /var/www/html/
   - Create index.html
   - Add basic HTML content  <h2> Good Morning , this is simple html web page </h2>

5. Access the Website
   - Copy the public IP of the EC2 instance
   - Open the IP in a browser using http://<public-ip>
```

Example 2 : 

```text
Use Case: Host a website using Nginx on an Ubuntu EC2 instance

1. Launch EC2 Instance
   - Region: N. Virginia (us-east-1)
   - AMI: Ubuntu
   - Instance type: t2.micro
   - Allow inbound ports: SSH (22), HTTP (80)

2. Connect to the EC2 Instance
   - Use SSH with the key pair

3. Switch to Root User : sudo -i / sudo su -

4. Install Nginx Web Server
   - Update packages --> apt update
   - Install nginx using apt or apt-get --> apt install nginx -y

5. Start and Verify Nginx Service
   - Start nginx --> systemctl start nginx
   - Check nginx service status --> systemctl status nginx

6. Configure Web Server
   - Navigate to cd /var/www/html/
   - Edit or create index.html cat>index.html and place the html content

7. Access the Website
   - Open browser
   - Visit http://<EC2-Public-IP>
```
