***Website:*** : is a Collection of web pages, and related content<br>
Types Of Website Hosting:
1. Static Website Hosting
2. Dynamic Website Hosting.

***1. Static Website Hosting:***<br>
A static website delivers to the user exactly what is stored, without any server-side processing.<br>
The Content of a static website is fixed and does not change.<br>
Examples: Amazon S3, GitHub Pages, etc...<br>
Example: E-Paper Website<br>

***STEP 1 – Create S3 Bucket (us-east-1)***
```
S3 → Create bucket
Bucket name: my-first-static-website-feb-12-26 (must be globally unique)
Region: US East (N. Virginia) – us-east-1
Uncheck Block all public access and Acknowledge warning
Create bucket
```

***STEP 3 – Enable Static Website Hosting***
```
Open your bucket → Properties → Static website hosting
Enable : Index document: index.html and Error document: error.html (optional) and Save changes
Copy the Website endpoint URL

Important: Website URL is HTTP, not HTTPS.
```

***STEP 4 – Add Bucket Policy (Make Website Public)**
```
Go to Permissions → Bucket Policy
Paste this (replace bucket name):

{
    "Version": "2012-10-17",		 	 	 
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": [
                "s3:GetObject"
            ],
            "Resource": [
                "arn:aws:s3:::Bucket-Name/*"
            ]
        }
    ]
}
```
