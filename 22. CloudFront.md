***CloudFront***<br>
```
. Content Delivery Network (CDN) by AWS
. Delivers content from nearest Edge Location to users
. Low latency & high speed for websites, APIs, videos
. Works with origins like Amazon S3, ALB, EC2, or custom HTTP servers
. Built-in security (HTTPS, AWS Shield, WAF integration)
. Caching reduces load on origin
. Pay-as-you-go pricing
. Supports origin failover
. Access logs & real-time metrics
. Supports signed URLs/cookies for private content
```

***Components***
```
| Component      | Purpose                         |
| -------------- | ------------------------------- |
| Edge Locations | Serve cached content near users |
| Distribution   | Main delivery configuration     |
| Origin         | Content source (S3/ALB/EC2)     |
| Cache Behavior | Control caching rules           |
| OAC            | Secure private S3 access        |
| WAF/Shield     | Protect against attacks         |
```
